# casino joao game

----------

This repo uses [backpack](https://github.com/palmerhq/backpack)

## How to use

Install it:
```bash
yarn
```
if you don't have yarn: [install yarn](https://yarnpkg.com/en/docs/install)


## How to run
run app for dev
```bash
yarn dev
```

run tests:
```bash
yarn test
```
